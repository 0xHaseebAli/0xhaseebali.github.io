Wazuh SIEM Deployment

Deployment of a home Security Operations Center using Wazuh as the SIEM platform, simulating a real-world attack, from initial reconnaissance to credential brute force and data exfiltration, and detecting it using Wazuh's security monitoring capabilities.

PART 01
Building the Lab
Wazuh setup, agent deployment, and network configuration
PART 02
Attack Simulation
Reconnaissance, brute force, SSH access, and data exfiltration
PART 03
Detection
Wazuh alert analysis, brute force detection, and post-exploitation investigation
PART 04
Incident Report
Attack timeline, findings, impact assessment, and recommendations
